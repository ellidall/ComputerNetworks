# Документация по UDP Pinger на C++

## Архитектура

Проект состоит из двух независимых компонентов, реализованных на **C++20** с использованием **POSIX-сокетов**:

### Клиент (udpPingerClient)
- Отправляет **10 UDP-пакетов** в формате: `Ping <sequence_number> <timestamp_ms>`.
- Устанавливает **таймаут 1 секунда** на ожидание ответа с помощью `setsockopt(SO_RCVTIMEO)`.
- Для каждого полученного пакета **проверяет его содержимое**: только если получено **точное совпадение** отправленного сообщения (по номеру последовательности и метке времени), ответ считается валидным.
- Вычисляет **RTT (Round-Trip Time)** с точностью до микросекунд через `gettimeofday()` и выводит в секундах с дробной частью (например, `0.000123 сек`).
- В случае отсутствия ответа или получения некорректного/чужого пакета — выводит `Request timed out`.

### Сервер (udpPingerServer)
- Работает как **эхо-сервер**: принимает любой UDP-пакет и отправляет его обратно **без изменения**.
- Не парсит и не интерпретирует содержимое — это позволяет клиенту самому проверять подлинность ответа.
- Поддерживает **автоматическое завершение** после **5 секунд бездействия** с помощью системного вызова `select()` с таймаутом.
- Безопасно проверяет входной порт: допускает только целые числа в диапазоне **1–65535**.
- Привязывается к `INADDR_ANY`, принимая запросы с любого сетевого интерфейса.
- Корректно освобождает сокет при завершении.

Оба компонента:
- Следуют требованиям ТЗ по именованию: функции — `UpperCamelCase`, переменные — `camelCase`, константы — `UPPER_CASE`.
- Гарантированно закрывают сокеты и не содержат утечек ресурсов.
- Завершаются с кодом `EXIT_FAILURE` при любых ошибках конфигурации или системных вызовов.

## Инструкция по сборке и запуску

### Требования
- Компилятор с поддержкой **C++20** (например, `g++` версии 11+ или `clang++` 14+).
- Операционная система: **Linux**, **macOS** или **WSL** (Windows без WSL не поддерживается).
- Установленный **CMake 3.22** или новее.

### Сборка
1. Убедитесь, что в корне проекта находятся файлы:
  - `CMakeLists.txt`
  - `udpPingerClient.cpp`
  - `udpPingerServer.cpp`
2. Выполните в терминале:
   ```bash
   mkdir build && cd build
   cmake ..
   make
